# Задача 1: `compose`

Напиши функцию **`compose`**, которая **склеивает функции справа налево**.

Раздел: замыкания, функции высшего порядка, композиция.

---

## Условие

```javascript
const add1 = (x) => x + 1;
const double = (x) => x * 2;

const fn = compose(add1, double);

fn(3); // 7
```

Почему `7`:

- сначала `double(3)` → `6`
- потом `add1(6)` → `7`

То есть: **`compose(f, g)(x) === f(g(x))`**

---

## Требования

1. `compose` принимает **несколько** функций: `compose(f, g, h, ...)`
2. Возвращает **одну** новую функцию
3. Эта функция принимает **один** аргумент `x`
4. Функции применяются **справа налево**

---

## Примеры для проверки

```javascript
compose(
  (x) => x + 1,
  (x) => x * 2
)(3); // 7

compose(
  (x) => x - 1,
  (x) => x * 2,
  (x) => x + 3
)(5);
// справа налево: +3 → 8, *2 → 16, -1 → 15
// ответ: 15
```

---

## Как организовать цепочку

### Две функции — вложенность

```javascript
compose(f, g)(x)  ===  f( g( x ) )
```

### Три функции

```javascript
compose(f, g, h)(x)  ===  f( g( h( x ) ) )
```

Схема:

```
x → h → g → f → результат
    ↑ сначала последняя в списке
```

### Много функций — цикл с конца

```javascript
function compose(...fns) {
  return function (x) {
    let result = x;

    for (let i = fns.length - 1; i >= 0; i--) {
      result = fns[i](result);
    }

    return result;
  };
}
```

На каждом шаге `result` — это «то, что уже прогнали через часть цепочки».

---

## `compose` vs `pipe`

| | `compose` | `pipe` |
|---|-----------|--------|
| Порядок | справа налево | слева направо |
| Пример | `compose(f, g)(x)` = `f(g(x))` | `pipe(g, f)(x)` = `f(g(x))` |

`pipe(f, g)(x)` === `compose(g, f)(x)`

---

## Частая ошибка

```javascript
return g(f(x)); // это pipe, не compose
```

Для `compose(f, g)` нужно **`f(g(x))`**, не наоборот.

---

## Решение

Файл практики: `interestedTasks/repeatMinStack.js`

```javascript
const add1 = (x) => x + 1;
const double = (x) => x * 2;
const fn = compose(add1, double);

function compose(...fns) {
  return function (x) {
    let result = x;

    for (let i = fns.length - 1; i >= 0; i--) {
      result = fns[i](result);
    }

    return result;
  };
}

console.log(fn(3)); // 7
```

---

## Что проверяет интервьюер

- Понимание **функции высшего порядка** (`compose` принимает функции, возвращает функцию)
- Умение построить **цепочку вызовов**
- Знание порядка: **последняя функция в аргументах вызывается первой**
