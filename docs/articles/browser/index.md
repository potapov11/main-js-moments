Материалы для фронтенд-собеседования: как работает браузер, сеть, асинхронность, безопасность.

## Планируется

- Event Loop (углублённо: call stack, microtasks, macrotasks)
- Promise API: `all`, `allSettled`, `race`, `any` (детальный разбор; кратко уже в async-await)
- CORS и `credentials: 'include'`
- Кэширование и заголовки HTTP
- DOM и рендеринг (краткий обзор)
- Web Storage, cookies, SameSite
