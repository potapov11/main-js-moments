## Уровень 0 — «руки помнят синтаксис» (~30 мин)

Пиши в `test.js`, без LeetCode.

| # | Задание | Проверь себя |
|---|---------|--------------|
| 0.1 | Вывести все элементы матрицы `3×3` двойным `for` | порядок: `1→2→3`, `4→5→6`, `7→8→9` |
| 0.2 | Для `grid` (2×3) вывести `m`, `n`, `m×n` | `2`, `3`, `6` |
| 0.3 | Создать матрицу `3×4` из нулей через `Array.from` | `matrix[1][2] === 0` |
| 0.4 | Скопировать матрицу через `map(row => [...row])`, изменить копию | оригинал не меняется |
| 0.5 | Заполни таблицу трассировки для `2×3` (как в доке) | 6 шагов, пары `(i, j)` |

**Критерий перехода:** не путаешь `matrix[i][j]` и `matrix[j][i]`; знаешь, что `length` строк ≠ `length` столбцов.

### Справка: m и n

| Обозначение | Что это | В коде |
|-------------|---------|--------|
| **m** | число строк | `matrix.length` |
| **n** | число столбцов | `matrix[0].length` |
| **m × n** | всего ячеек | `m * n` |

## Уровень 1 — строки, столбцы, агрегаты (~1 день)

Матрица для примеров:

```javascript
const nums = [
  [1, -2, 3],
  [4,  5, 6],

```

| # | Задание | Ожидаемый ответ |
|---|---------|-----------------|
| 1.1 | Сумма всех элементов | `17` |
| 1.2 | Максимум | `6` |
| 1.3 | Количество отрицательных | `1` |
| 1.4 | Сумма строки `i = 1` | `15` |
| 1.5 | Сумма столбца `j = 0` | `5` |
| 1.6 | Новая матрица: каждый элемент × 2 | `[[2, -4, 6], [8, 10, 12]]` |
| 1.7 | Массив сумм каждой строки | `[2, 15]` |

**Критерий:** отдельно пишешь цикл по строке (`j` бежит) и по столбцу (`i` бежит).

## Уровень 2 — углы, соседи, диагонали (~1 день)

Матрица `3×3` с числами `1..9`:

```javascript
const matrix = [
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9],
];
```

| # | Задание |
|---|---------|
| 2.1 | Вернуть массив 4 углов |
| 2.2 | Для `(i, j)` вернуть всех валидных соседей (4 направления) |
| 2.3 | Сумма главной диагонали (`matrix[i][i]`) |
| 2.4 | Сумма побочной (`matrix[i][n - 1 - i]`) |
| 2.5 | Сумма обеих диагоналей (центр не считать дважды) |

**Мини-тест:** для `(0, 0)` сколько соседей? для `(1, 1)`?

- [ ] Двойной цикл без подглядывания

- [ ] `m = matrix.length`, `n = matrix[0].length`
- [ ] Соседи с проверкой границ
- [ ] Главная и побочная диагональ
- [ ] Transpose на бумаге (`result[j][i] = matrix[i][j]`)

## Пять паттернов (шпаргалка)

| Паттерн | Суть |
|---------|------|
| Обход `i, j` | два вложенных цикла, O(m×n) |
| Диагонали | главная: `matrix[i][i]`; побочная: `matrix[i][n-1-i]` |
| Transpose | `result[j][i] = matrix[i][j]` |
| Spiral | `top`, `bottom`, `left`, `right`, сужать рамку |
| Группы (Sudoku) | Set на строку / столбец / блок |
