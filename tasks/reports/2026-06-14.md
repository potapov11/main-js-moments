## Что прошли

### 1. 26. Remove Duplicates from Sorted Array

**Статус:** ✅ решено и разобрано

**Решение:** два указателя `read` / `write`

- `write = 0`, цикл с `read = 1`
- если `nums[read] !== nums[write]` → `write++`, записываем `nums[read]`
- возвращаем `write + 1`

**Сложность:** O(n) время, O(1) память

**Ключевые выводы:**

- функция возвращает `k`, но также **мутирует** `nums`: первые `k` элементов — уникальные
- работает только потому, что массив **отсортирован** (дубликаты идут блоками)
- инвариант: `nums[0..write]` — уникальные элементы в исходном порядке

**Дополнительно:** рассмотрен вариант через `Set` (ES6)

- считает количество уникальных — O(n) время
- память O(k), не O(1)
- **не подходит** для задачи 26: массив `nums` не изменяется in-place
- полезен для задач типа 217 (Contains Duplicate), где нужен только факт/количество

### 2. 283. Move Zeroes

**Статус:** ✅ решено (два варианта)

#### Вариант A — два цикла (copy + fill)

- 1-й цикл: `read` проходит массив, ненулевые пишем в `write`, `write++`
- 2-й цикл: `nums[write..n-1] = 0`

**Сложность:** O(n) + O(n) = O(n) время, O(1) память

#### Вариант B — один цикл (swap)

- `write = 0`, `read` проходит массив
- если `nums[read] !== 0` → swap `nums[write]` и `nums[read]`, `write++`

**Сложность:** O(n) время, O(1) память

**Ключевые выводы:**

- тот же паттерн read/write, что и в задаче 26
- условие записи: «элемент не ноль»
- swap с **двух концов** (`start`/`end`) для этой задачи **не подходит** — ломает порядок ненулевых
- по сути это 27. Remove Element (убрать 0) + добить хвост нулями (или swap)

## Ошибки и исправления

| Момент | Ошибка | Исправление |
|--------|--------|-------------|
| Move Zeroes (swap) | `write` = позиция последнего нуля | `write` = куда положить **следующий** ненулевой |
| Move Zeroes (swap) | `write` не увеличивался после записи | после swap / записи всегда `write++` |
| Set для задачи 26 | считали только `.size`, массив не менялся | для in-place нужен read/write |
