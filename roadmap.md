- Многомерные массивы

3. **27. Remove Element** - Удалить все вхождения элемента +
4. **35. Search Insert Position** - Найти позицию для вставки элемента +
5. **53. Maximum Subarray** - Найти максимальную сумму подмассива +
6. **66. Plus One** - Увеличить число, представленное массивом, на 1 --
7. **88. Merge Sorted Array** - Объединить два отсортированных массива
8. **121. Best Time to Buy and Sell Stock** - Максимальная прибыль от покупки/продажи акций++
9. **169. Majority Element** - Найти элемент, встречающийся более n/2 раз
10. **217. Contains Duplicate** - Проверить наличие дубликатов+
11. **283. Move Zeroes** - Переместить все нули в конец массива
12. 152. Maximum Product Subarray

## 📚 Дополнительные ресурсы

### Полезные материалы:

- **Big O Notation** - Изучите сложность алгоритмов
- **LeetCode Explore** - Интерактивные уроки по темам
- **NeetCode** - Структурированные плейлисты по темам
- **AlgoExpert** - Видео-объяснения решений

### Рекомендации по изучению:

1. **Начните с массивов и строк** - это основа
2. **Изучайте по 1-2 темы в неделю**
3. **Решайте минимум 2-3 задачи в день**
4. **Пишите код от руки** на доске/бумаге
5. **Объясняйте решение вслух** - как на собеседовании
6. **Повторяйте решенные задачи** через неделю

### Структура решения задачи:

1. **Понять задачу** - перефразировать своими словами
2. **Придумать примеры** - edge cases
3. **Придумать подход** - алгоритм
4. **Оценить сложность** - время и память
5. **Написать код**
6. **Протестировать** - на примерах
7. **Оптимизировать** - если возможно
