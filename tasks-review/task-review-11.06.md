## Задача 2 — ✅ Сделана

**349. Intersection of Two Arrays** (через `Set`)

### Решение
```javascript
const setArray = new Set(nums1);
const resultArr = new Set();
for (...) {
  if (setArray.has(nums2[i])) resultArr.add(nums2[i]);
}
return [...resultArr];
```

### Что исправлялось
- Был захардкожен `new Set([4,9,5])` → заменён на `new Set(nums1)`.
- `resultArr` переведён с массива + дедупликации на `Set` сразу.

### Оценка
| Критерий        | Результат |
|-----------------|------------|
| Корректность    | ✅         |
| Уникальность    | ✅         |
| Время O(m + n)  | ✅         |
| Память          | O(m + k)   |

**Уровень:** уверенное решение.
