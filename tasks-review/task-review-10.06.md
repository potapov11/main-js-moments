## Задача 2 — ✅ Сделана

**344. Reverse String**

### Решения
1. Классический swap через `temp` + `while (start < end)`.
2. Синтаксический вариант с деструктуризацией: `[a, b] = [b, a]`.

### Оценка
| Критерий        | Результат |
|-----------------|-----------|
| Корректность    | ✅        |
| In-place        | ✅        |
| O(1) доп. памяти | ✅       |
| Сложность       | O(n) время |

**Уровень:** уверенный middle.

### Плюс
- Понял разницу между O(1) доп. памяти и O(n) (новый массив / копия).
- Сравнил два стилистических варианта, оба приняты.

## Задача 3 — ✅ Сделана

**125. Valid Palindrome**

### Что было
- Верная структура: два указателя, пропуск мусора, сравнение в нижнем регистре.
- Баг: `isLetter` через `toLowerCase() !== toUpperCase()` **пропускала цифры**.
- Пример `"0P"` возвращал `true` вместо `false`.
- После разбора: `isLetterAndIsNum` (буква **или** цифра) + `start < end` во внутренних циклах.

### Итоговое решение
- `isChar || isNum` (charCode 48–57 для цифр).
- Защита указателей при пропуске небуквенно-цифровых символов.

### Оценка
| Критерий        | Результат |
|-----------------|-----------|
| Корректность    | ✅        |
| Буквы + цифры   | ✅ (после фикса) |
| Edge cases      | ✅ (`" "`, `"0P"`) |
| Сложность       | O(n) время, O(1) память |

**Уровень:** уверенный middle.
