# Daily interview feedback

Один файл на календарный день: `interview-feedback-DD.MM.md`.

Агент пишет сюда после **каждого** закрытого вопроса mock-собеседования.

Шаблон записи — в `.cursor/cursorrules/.cursorrules` (раздел «Фиксация прогресса»).

Слабые вопросы дублируй **с полной формулировкой** — для самостоятельного повтора.
